Lists in HTML

Ordered List

Ordered lists are used where items need to be in a specific order. Changing order or items might result in wrong sequence. Examples are: Algorithms, Recipes, Index of a book etc.

  4. That's it for checking size of the disk

Unordered List

Unordered lists are used where order of items does not matter. Changing order of items will still result in same meaning. Examples are: Shopping List, books, games list etc.

Description List

Description lists are used when we need to describe list of terms Order of items is does not matter, its your personal choice. Example: A dictionary, Theseraus, Terminologies

Computer Terminologies

Hardware

Processor
A electronic device that has ability to perform calculations on zeros and ones using a specific set of instructions, known as Instruction Set
